3DO INTERACTIVE MULTIPLAYER (3DO) — a UD0 EXEREÚNESIS teardown + spec edition. The 3DO was a specification, not a single product: Trip Hawkins's company licensed a CD-based design that Panasonic, Goldstar, and others built. An ARM CPU with twin custom video coprocessors made it powerful for 1993 — and a 699-dollar launch price made it a commercial cautionary tale. Render-not-invent, two-layer (datasheet fact vs lore); no ROMs/BIOS/code; manufacturer cited not minted. LIVE clickable block-diagram teardown + spec sheet. Vellum (no black) + quorum-ring favicon. 6 components.
